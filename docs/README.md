# Documentation

This directory contains project documentation for the Unicorn Weather Index.

## Contents

- API documentation
- Architecture decisions
- Setup and deployment guides
- User guides

## Project Overview

The Unicorn Weather Index is a data visualization dashboard that compares individual GitHub activity against aggregated data from major Bangalore tech unicorns, correlated with local weather patterns.

## Key Features

- Real-time GitHub activity analysis
- Weather correlation insights
- Professional fintech-style dark theme
- Responsive design for all devices
- "David vs. Goliath" productivity comparisons
